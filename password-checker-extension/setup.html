Now complete the setup to enable secure local file storage.

1 Install Python (if not already installed)

This extension requires Python 3.7+ to save passwords as local files.

  • Download from python.org
  • Make sure to check "Add Python to PATH" during installation
  • Verify installation by opening Command Prompt and typing: python --version

2 Run the Registration Script

Navigate to the native-host folder and run the registration script:

cd "d:\password strength\native-host"
register.bat

This will register the native messaging host with Chrome.

3 Test the Extension

Visit any website with a login form and try entering a password. You should see:

  • Real-time password strength analysis
  • Character-by-character breakdown
  • Security suggestions
  • Password reuse warnings
